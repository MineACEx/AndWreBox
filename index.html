⚡ 功能特性

15 大模块，覆盖 Android Root 工具全场景

01
系统微调
调整 CPU 调度、GPU 频率、I/O 调度器。支持一键性能模式、省电模式、游戏模式预设。
02
温控调节 NEW
自定义温控阈值，解除温控限制。支持逐场景配置（充电/游戏/日常），安全区间保护。
03
AnyKernel3 刷写 BETA
内置答题验证系统，通过后解锁刷写界面。支持自动备份 boot.img，安全刷入自定义内核。
04
Magisk 模块构建
一键生成改机型模块，预设 36 款热门机型。支持自定义模块名称、版本号、作者信息。
05
电池健康监测
实时读取电池循环次数、健康度、温度。支持充电上限设置，延长电池寿命。
06
GPU 优化
调整 GPU 最低/最高频率，锁定渲染帧率。支持 Adreno 全系列，含降压超频选项。
07
网络优化
调整 TCP 拥塞算法、DNS 配置、MTU 参数。降低游戏延迟，提升下载速度。
08
显示调节
调整 DPI、分辨率、刷新率、色温。支持自定义分辨率和 HDR 强制开启。
09
ZRAM / SWAP 管理
调整 ZRAM 大小和压缩算法（lz4/zstd）。优化内存使用，减少杀后台概率。
10
熵池增强
增大随机数熵池，提升系统响应速度和随机性。对加解密操作有明显加速效果。
11
Prop 编辑器
可视化编辑 build.prop 和系统属性。内置常用优化项预设，一键恢复默认值。
12
实时监控
显示 CPU/GPU 频率、温度、帧率、电池功耗。悬浮窗模式，游戏中实时查看。
13
文件加密
AES-256 加密任意文件/文件夹。支持密码和解锁图案两种方式，加密后自动隐藏。
14
应用管理
批量冻结/解冻系统应用，禁用/启用组件。支持导出应用列表，一键恢复预设配置。
15
备份还原
备份和还原系统分区、应用数据、内核镜像。支持 TWRP / Orangefox 兼容格式。

📦 安装教程

两种方式，任选其一

A

方式一：Termux 安装

适用于已安装 Termux 的用户

pkg update && pkg install python -y
cd /sdcard/AndWreBox
python main.py
B

方式二：Python3 模块安装

刷入模块后无需 Termux 即可运行

# 1. 下载 Python3Install.zip 并在 Magisk/KernelSU 中刷入
# 2. 重启设备
# 3. 执行以下命令
cd /sdcard/AndWreBox
python3 main.py
1

下载并解压

从下载页面获取最新版本，将 AndWreBox 文件夹解压到 /sdcard/AndWreBox/

2

安装 Python 环境

选择上述方式一或方式二安装 Python 运行环境

3

运行脚本

进入 AndWreBox 目录，执行 python main.py（或 python3 main.py），根据菜单选择功能
